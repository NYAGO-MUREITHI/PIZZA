# PIZZA-PALACE

## AUTHOR
 NYAGUTHIE MUREITHI

## DESCRIPTION
This project is a web-application used to order pizza online from a restaurant known as pizza palace where the user gets to pick the flavor, quantity and size of pizza that they pick and choose whether they want a delivery or pick-up.

## TECHNOLOGY USED
The project was written in HTML language and edited using .css style-sheet. It also uses JavaScript scripting language. It was edited on atom.

## SETUP INSTRUCTIONS
The website has been run on Chrome without any challenges . 
1. Download Google Chrome on https://www.google.com/chrome/
2. Click on the live link provided below to run the website
